# Stack Tecnológico – Movie Agenda

## Fase 1 (Actual)

### Lenguajes Base
- HTML5 (Estructura semántica).
- CSS3 (Estilos personalizados).
- JavaScript Vanilla (Lógica e interactividad).

### Control de Versiones
- Git.
- GitHub.

### API Externa
- The Movie Database (TMDB API).
- Consumo mediante fetch().
- Manejo de respuestas JSON.

### Almacenamiento
- LocalStorage para guardar la agenda del usuario.

---

## Fase 2 (Interactividad Avanzada)

- Manejo dinámico del DOM.
- Filtros por categoría.
- Buscador en tiempo real.
- Mejor manejo de estados.

---

## Fase 3 (Escalabilidad y Profesionalización)

### Frontend
- React.
- Componentización:
  - MovieCard
  - MovieList
  - Playlist
  - Navbar

### Base de Datos
- Firebase Firestore.

### Autenticación
- Firebase Authentication.
